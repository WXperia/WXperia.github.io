【前端|ES6】Set与Map的区别

Set 与 Map 是 ES6 规范中提供的新数据结构。

  • Set 类似于数组，但是成员的值都是唯一的，没有重复的值。

使用

  • Set 可以接收一个数组，用于初始化
  • Set 也可以接收一个字符串
  • Set 添加值时，不会发生类型转换 5 和 '5' 是不同的值，这样的比较类似于 === 唯一的区别就是 Set 加入值时，认为NaN等于自身，而 === 认为 NaN 不等于自身。
